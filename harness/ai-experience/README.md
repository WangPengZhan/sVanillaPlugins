# AI Experience

Use this section for lessons learned while AI agents work on the repository.

Good entries include:

- recurring failure modes and how to avoid them;
- prompts or workflows that produced reliable results;
- agent handoff notes that are still useful after the original task;
- examples of good verification scope for common changes.

Avoid storing:

- temporary chain-of-thought or raw scratch notes;
- secrets, credentials, cookies, or private service data;
- agent-specific commands that do not generalize.
